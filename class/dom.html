//dom-->document object model //it is a programming interface for web documents //it represents the page so that programs can change the document structure ,style and content //it represents the page as nodes and objects //that way programming languages can connect to the page //the document object is the root node of the html document //we can use javascript to access and manipulate the document object //we can use document object methods and properties to access html elements //we can use document.getelementbyid to access an element by its id //we can use document.queryselector to access the first element that matches a css selector //we can use document.queryselectorall to access all elements that match a css selector //we can use document.getattribute to get the value of an attribute of an element //we can use document.setattribute to set the value of an attribute of an element //we can use document.style to change the style of an element //we can use document.innerhtml to change the content of an element //we can use document.textcontent to change the text content of an element
