//Middleware

//1.Why we use middleware ?
To process request before sending response
ex.logging,authentication

//2.what are the types of middleware?
1. Application-Level Middleware

 Works on the entire application
 Runs for every request
 Used for logging, authentication, etc.

2. Router-Level Middleware

 Works only for specific routes
 Applied to a particular group of URLs
 Helps in organizing code better

3. Built-in Middleware

 Already provided by Express
 Used for common tasks like handling JSON data or static files

4. Third-Party Middleware

 Installed from external packages
 Adds extra features like security, CORS, etc.

5. Error-Handling Middleware

Used to handle errors in the application
Prevents app from crashing and sends proper error response

//3.What are the benefits of middleware ?
Reuse code
Better security
Clean structure

//4.Where the middleware is used in company level ?
Authentication (JWT)
Logging (user activity)
Security checks
API validation
